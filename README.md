# Gyroscope Ball Game

An Android game built with Jetpack Compose that uses the device's accelerometer to control a ball through a maze. Tilt your phone to move the ball, avoid the walls, and reach the green goal area to win.

---

## Features

- Tilt-based movement using the device accelerometer  
- Maze walls and collision detection  
- Win condition when reaching the goal area  
- Start screen and restart option  
- Movements updated every 16ms  

---
## How to Run
1. Clone this repository:
   ```
   git clone https://github.com/shanji361/GyroscopeBallGame.git
   ```
2. Open the project in Android Studio.

3. Run the app on an emulator or a physical Android device.
